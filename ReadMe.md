# CoupangPartnersWeb - 개발중

## 프로젝트 소개

-   기술 스택 : Typescript , React, NextJS, StyledComponents, Ant-Design, NodeJs
-   배포 : vercel

-   웹서비스에 대한 자세한 개요
    - 쿠팡파트너스 활동을 위한 웹 서비스 입니다.
    - 쿠팡파트너스 API를 활용해 제품 정보를 받아서, 포스팅합니다.
    - 전자제품, 가전제품, 패션 및 잡화 등의 인기추천 순위를 보여줍니다.

## 주요 기능

- 검색어 자동완성
    - downshift 라이브러리 사용
    - antd Input 사용
    
- 동젝페이지 생성
    - 상품항목별 동적페이지 생성
   
- 카카오톡 공유하기
